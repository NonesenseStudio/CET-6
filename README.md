# 大学英语六级 (CET-6) 真题资料

大学英语六级考试（CET-6）历年真题合集，涵盖 **2015 年 6 月 ~ 2025 年 12 月** 共 **25 场**考试的全套备考资料。

## 📦 内容概览

每场考试包含以下资料：

| 资料类型 | 说明 |
|---------|------|
| **真题 PDF 版** | 高清扫描版真题，推荐打印练习 |
| **真题 Word 版** | 可编辑的 docx 格式真题（2015~2023） |
| **答案解析** | 每套真题的详细答案解析 PDF |
| **听力音频** | 对应各套真题的听力 MP3 音频 |

> **文件统计**：共 578 个文件，含 149 份 PDF、62 份 Word、47 段 MP3 音频。

## 📊 阅读分析

项目附带 **[CET6阅读分析总结](CET6阅读分析总结.md)**，基于 69 份阅读文本的系统分析：

- **高频词汇**：`data`、`consumer`、`researcher`、`online` 等为六级高频核心词
- **文章题材**：科技/互联网 (20.9%)、经济/商业/就业 (17.9%)、教育/学习/心理 (17.2%)、环境/生态 (13.4%) 四大类占比近七成
- **题型分布**：细节理解题占绝对主导 (81.8%)，推断判断 (8.5%)、态度评价 (6.9%) 次之

## 🚀 使用建议

1. **刷真题**：优先使用 PDF 版打印练习，模拟真实考试环境
2. **精读解析**：做完题后对照答案解析，重点分析错题原因
3. **听力训练**：配合 MP3 音频反复练习，适应六级听力语速和题型
4. **词汇积累**：结合阅读分析报告中的高频词汇有针对性地记忆
